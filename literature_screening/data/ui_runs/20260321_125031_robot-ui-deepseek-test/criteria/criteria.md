# Topic

面向地下基础设施物联网的自挖掘/地层机器人（Self-burrowing/Earth-penetrating Robots）机制分类与应用

## Inclusion

- 必须涉及能够主动向地下、土壤或颗粒介质中钻探、挖掘、穿透的机器人技术（如蠕动、旋转、挖掘、生长、流化机制）
- 或重点探讨此类地层机器人在地下环境中的定位、通信协同及物联网（IoUT）集成

## Exclusion

- 星际与太空探索研究（如针对月壤、火星土壤等外星壤的深空挖掘机器人）
- 纯生物学或农业研究（仅研究动植物发育，无机器人应用）
- 传统大型土木工程盾构/挖掘设备（无小型化、智能化特征）
