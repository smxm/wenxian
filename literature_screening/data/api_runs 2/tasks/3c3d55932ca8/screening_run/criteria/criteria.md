# Topic

弓网离线电气特性、电弧模型（尤其是Cassie及其衍生模型）及MATLAB弓网仿真

## Inclusion

- 必须涉及电气化铁路/高铁的弓网系统 Pantograph-catenary system
- 核心内容必须探讨弓网离线 Loss of contact/Arcing 引发的电气特性变化 如电压/电流畸变、谐波等
- 包含电弧的数学/物理建模 如Cassie、Mayr等黑盒模型 或等效电路的计算机仿真 如MATLAB/Simulink 。

## Exclusion

- 纯机械动力学或受流质量研究 完全无电气分析
- 非弓网场景的电弧研究 如焊接、断路器电弧，且未说明可移植到弓网
- 缺乏具体模型公式或仿真/实验波形支撑的纯定性探讨
- 非中英文文献。
