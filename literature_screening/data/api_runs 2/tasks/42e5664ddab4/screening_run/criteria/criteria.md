# Topic

以“研发支出比率（研发强度）”作为“企业风险承担”代理变量的实证与理论研究

## Inclusion

- 必须在研究设计或理论框架中，明确将“研发支出 R&D ”或“研发强度”作为衡量“企业风险承担 Risk-taking ”水平的指标或代理变量。
- 探讨了影响企业研发风险承担的前因变量（如政策、高管特质、公司治理等），或其带来的经济后果（如企业价值、创新绩效等）。
- 属于经济学、管理学、公司财务或战略管理领域的高质量实证研究或综述。

## Exclusion

- 纯自然科学或工程技术类的研发文献 如具体的技术攻关细节，无管理学/经济学视角 。
- 虽然研究了研发投入，但并未将其与“风险承担/风险偏好”概念建立理论联系的常规文献。
- 仅使用盈余波动率或股票回报率波动衡量风险承担，且全文未涉及研发支出指标的文献。
