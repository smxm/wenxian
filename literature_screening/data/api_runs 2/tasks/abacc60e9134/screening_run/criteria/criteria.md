# Topic

基于BIM技术的某项目施工图预算

## Inclusion

- 必须涉及 BIM 技术（特别是 5D BIM）在项目造价、成本估算、工程量算量（QTO）或施工图预算编制中的直接应用。
- 研究内容包含具体项目的应用案例、传统预算与BIM预算的对比（效率、准确率）、或造价数据互操作性（如 IFC 标准、Revit 与造价软件的数据交互）。
- 属于高质量的实证研究、案例分析或系统性综述。

## Exclusion

- 仅停留在 3D 建模、碰撞检查或 4D 进度管理，未实质性涉及造价成本（5D）数据的文献。
- 侧重于施工阶段后的设施运维（FM）、结构力学分析或纯建筑设计的文献。
- 无实际工程背景依托的纯理论算法推导，或非建筑/土木工程领域的文献。
