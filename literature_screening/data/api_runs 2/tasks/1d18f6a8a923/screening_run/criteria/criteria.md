# Topic

企业数字档案系统建设与数据化开发利用

## Inclusion

- 研究对象必须是企业、政府或大型机构的档案/电子记录。
- 核心内容必须涉及以下至少一项：数字档案系统架构设计、电子文件全生命周期管理（前端控制/业务协同）、档案数据安全与长期保存（如OAIS模型）、档案数据化与知识管理（如AI提取、知识图谱）。
- 必须是具有具体实施方案、模型框架或高质量综述的学术文献。

## Exclusion

- 纯实体档案物理保护（如纸张去酸、库房温湿度、虫害控制）。
- 纯图书馆学或公共文化馆藏管理（非组织机构的业务档案）。
- 脱离档案管理实际业务的纯底层IT算法测试。
