# Topic

成人白血病患者赋能教育（Empowerment Education）干预与应用

## Inclusion

- 研究对象必须是成人白血病患者（纯患者群体）。
- 核心干预措施或研究主题必须涉及“赋能教育”（包括授权教育、自我效能提升、自我管理健康教育等）。
- 文献发表时间必须在2016年及以后。

## Exclusion

- 研究对象包含儿童、婴幼儿或未成年人。
- 研究/干预对象包含患者家属、照顾者、父母（必须是针对患者本人的研究）。
- 纯临床病理机制、药物疗效试验或非教育/非心理干预类的医学研究。
- 发表时间在2016年以前。
