# Topic

互联网+在老年慢性病患者中应用的护理

## Inclusion

- 必须包含“互联网/信息化技术”载体 如 : 移动医疗、微信、App、物联网、远程监测等
- 研究对象必须明确为“老年人”或高龄人群；
- 疾病类型必须为“慢性病”（或特指的 [此处填入具体病种]）；
- 核心内容必须属于“护理干预”、“延续性护理”、“自我管理”或“健康教育”范畴，且有明确的护理结局指标。

## Exclusion

- 研究对象为非老年人群（如儿童、中青年）；
- 非慢性病（如急性创伤、突发传染病）；
- 纯计算机算法、系统架构开发或纯硬件工程研究（缺乏真实的临床或社区护理应用数据）；
- 纯医学治疗、病理机制或药理研究（无护理/管理干预成分）；
- 仅为传统常规护理，未结合任何“互联网+”或信息化手段。
