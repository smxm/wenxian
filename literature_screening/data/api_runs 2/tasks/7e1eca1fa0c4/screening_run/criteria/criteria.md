# Topic

游戏化教学法在音乐鉴赏课中的运用

## Inclusion

- 核心交叉：必须同时涉及“游戏化教学/教育游戏/游戏化机制”与“音乐教育/音乐鉴赏/音乐欣赏”两个领域。
- 机制与应用：优先纳入探讨了具体游戏化机制（如积分、闯关、角色扮演、即时反馈等）如何在音乐课堂中落地的文献。
- 效果验证：优先纳入包含实证研究、教学实验数据或心流理论/学习动机理论探讨的文献。

## Exclusion

- 纯音乐本体研究：仅分析音乐史料、曲式结构、作曲技法，完全不涉及教学法或受众心理的文献。
- 纯技术开发：仅讨论音乐游戏软件的代码编写或算法逻辑，缺乏教育教学场景应用的文献。
- 非音乐领域：探讨游戏化教学，但应用学科为语数外等非音乐学科的文献。
