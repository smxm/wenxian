# Topic

Step 1: 筛选判读

## Inclusion

- 必须明确涉及人工智能技术（如机器学习、生成式AI、计算机视觉、行为树等）。
- 必须涉及 XR 技术（VR、AR、MR）或元宇宙、3D虚拟空间。
- 核心研究对象或应用场景必须包含动物（特别是猫咪/Feline）或虚拟宠物（Virtual Pets/Companions）。
- 重点关注“交互”层面（如人宠互动、动物行为分析、虚拟伴侣的情感反馈）。

## Exclusion

- 纯兽医学或动物生物学研究，无前沿计算机技术（AI/XR）介入。
- 纯 AI 算法理论或纯 XR 硬件研究，未结合动物交互或虚拟宠物场景。
- 传统的2D宠物游戏开发（无 AI 自主性或 XR 沉浸感）。
- 非英语或非学术类文章（如纯商业新闻报道）。
