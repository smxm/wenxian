# Topic

隐球菌侵袭血脑屏障的深层机制及体外仿生血脑屏障模型构建

## Inclusion

- 研究对象必须强相关“隐球菌（Cryptococcus）”与“血脑屏障（BBB/脑微血管内皮细胞等）”的互作；或专门针对“高级血脑屏障模型（微流控、器官芯片、iPSC衍生等）”的构建与应用。
- 必须是探讨深层分子机制（如穿胞作用、紧密连接破坏、特洛伊木马机制）或新型仿生模型开发的高质量研究/综述。

## Exclusion

- 仅涉及隐球菌肺部感染或其他器官感染，未深入探讨血脑屏障的文献。
- 纯常规药物疗效对比、流行病学统计或缺乏机制探索的单一临床病例报告。
- 传统的二维静态细胞培养且无明显方法学创新的研究。
