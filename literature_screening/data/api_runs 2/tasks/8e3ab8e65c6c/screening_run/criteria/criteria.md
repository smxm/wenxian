# Topic

基于深度学习的冷轧带钢表面缺陷检测与分析

## Inclusion

- 研究对象必须是冷轧带钢、带钢或高度相关的金属表面缺陷。
- 核心技术必须涉及前沿深度学习/计算机视觉方法（如：无监督异常检测、小样本/长尾学习、域自适应/泛化、轻量化边缘网络、或多模态根因分析）。
- 具有明确的工业落地痛点解决思路或算法创新。

## Exclusion

- 纯传统的冶金机理、机械结构或材料化学成分研究，无视觉/数据驱动算法介入。
- 过于陈旧的传统图像处理方法（如单纯的阈值分割、传统边缘算子），缺乏深度学习或现代机器学习背景。
- 与工业缺陷检测无关的纯理论算法推导。
- 非中英文文献。
