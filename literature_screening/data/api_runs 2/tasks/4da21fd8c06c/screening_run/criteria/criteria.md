# Topic

声测管倾斜管距修正算法与模型

## Inclusion

- 必须涉及超声波透射法（Crosshole Sonic Logging, CSL）、超声波检测或声波CT层析成像。
- 必须涉及基桩、深基础或混凝土结构的完整性检测。
- 研究内容必须包含误差修正或信号处理。具体包括但不限于：空间轨迹计算、管距/距离补偿、几何模型拟合（直接法）；或小波变换、神经网络、异常特征剔除等数据处理算法（间接法）。

## Exclusion

- 纯粹的基桩承载力测试（如静载试验、高应变法）或施工机械研究。
- 纯混凝土材料化学配比、水化热或力学性能研究（未涉及超声无损检测）。
- 完全没有提及数据修正、误差补偿或相关数学/机器学习算法的纯基础操作报告。
