# Topic

人工智能驱动下的智慧物流仓储系统优化

## Inclusion

- 研究内容必须直接涉及“智慧物流”的核心技术应用（如：物联网IoT、人工智能AI、大数据、数字孪生、区块链等）或其管理学影响（如：供应链韧性、绿色可持续发展）。
- 必须有具体的物流应用场景落地（如：仓储、配送、冷链、港口等）。
- 文献类型需为高质量的实证研究、案例分析或系统性综述。

## Exclusion

- 纯传统物流或供应链管理研究（缺乏数字化、智能化技术要素）。
- 纯计算机底层算法理论或纯硬件研发（未结合物流业务场景进行验证）。
- 缺乏实质性数据支持的低质量短文或非学术性新闻报道。
