# Topic

左心室辅助装置（LVAD）术后目标平均动脉压（MAP）的管理策略及其与临床预后的关系

## Inclusion

- 研究对象必须是植入左心室辅助装置（特别是连续血流型 CF-LVAD，如 HeartMate 系列、HVAD 等）的成人患者。
- 核心变量必须包含术后平均动脉压（MAP）的管理、目标范围设定、测量技术（如多普勒、动脉导管）或血压控制药物。
- 必须涉及 MAP 与临床预后的关联（如中风、泵血栓、主动脉瓣关闭不全、胃肠道出血或生存率）。

## Exclusion

- 纯探讨外科植入手术技术或器械机械设计的文献（不涉及术后血流动力学管理）。
- 仅针对右心室辅助装置（RVAD）或全人工心脏（TAH）的研究。
- 动物实验或体外模拟（Mock loop）实验（除非对临床指南有直接奠基性作用）。
- 纯儿科（Pediatric）病例研究。
