# Topic

前沿超声多模态技术（超微血流SMI/MVFI、粘弹性成像SWE、超分辨率超声造影SR CEUS）在妇科生殖（尤其是子宫内膜蠕动波及微血流评估）中的应用

## Inclusion

- 研究对象必须涉及妇产科或生殖医学领域（如子宫、子宫内膜、卵巢、盆腔微环境等）。
- 核心技术必须包含至少一种前沿超声技术：超微血流（SMI/MVFI/超快多普勒）、粘弹性成像（SWE/剪切波弹性）、超分辨率超声造影（SR-CEUS/ULM），或重点探讨了子宫内膜蠕动波（Uterine peristalsis）。
- 研究类型包含临床试验、高质量动物模型研究或权威综述。

## Exclusion

- 纯超声仪器硬件开发或纯物理算法理论，缺乏医学/生物学应用场景。
- 研究器官非妇科生殖系统（如单纯研究肝脏、乳腺、甲状腺等，除非其方法学可直接迁移至妇科应用）。
- 仅使用传统二维超声或常规彩色多普勒，未涉及上述前沿技术。
