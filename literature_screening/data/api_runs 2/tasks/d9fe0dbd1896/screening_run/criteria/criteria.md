# Topic

数字普惠金融对城乡收入差距的影响

## Inclusion

- 必须明确涉及数字金融/金融科技变量（如FinTech, Mobile Money, Digital Credit, 移动支付等）。
- 必须探讨了对收入不平等、财富差距、减贫或城乡双层结构的经济学影响。
- 侧重于探讨机制（如信贷约束缓解、风险分担）、异质性（数字鸿沟）或空间溢出效应的实证或高质量理论研究。

## Exclusion

- 纯计算机科学或密码学视角的区块链/算法研究（无微观或宏观经济后果探讨）。
- 仅探讨传统金融（无数字/技术属性）或传统普惠金融，且未将其作为对比基准的文献。
- 与收入、贫困、不平等或城乡要素流动毫无关联的纯公司金融研究（如单纯讨论企业股价、高管薪酬）。
