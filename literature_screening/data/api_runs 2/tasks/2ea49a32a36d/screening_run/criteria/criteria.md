# Topic

机器学习在场地效应（Site Effects）评估与预测中的应用

## Inclusion

- 核心方法：必须包含具体的机器学习或深度学习技术（如神经网络、随机森林、XGBoost、替代模型等）。
- 研究目标：必须明确针对“场地放大（Site Amplification）”的量化评估或预测。例如：预测放大系数（Amplification Factors, AF）、模拟频率依赖的放大谱、非线性土层放大响应、地形放大或盆地放大效应。
- 数据基础：基于实际地震动记录数据（强震台网数据）或经过验证的大规模数值模拟生成的数据集进行模型训练。

## Exclusion

- 偏离“放大”目标：仅进行粗略的场地类别划分（Site Classification），或仅预测 $V_{S30}$ 等参数但未将其用于计算场地放大。
- 偏离“场地”本身：研究对象集中在地表建筑结构（上部结构）的抗震损伤评估、减震隔震设计，而非下方土体/地形的放大效应。
- 传统方法：纯经验公式拟合或纯传统一维/二维等效线性化物理模拟，未使用机器学习算法。
- 纯算法理论探讨无实际工程地震学背景，或非英语文献。
