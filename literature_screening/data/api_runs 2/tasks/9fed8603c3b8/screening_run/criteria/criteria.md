# Topic

隐球菌侵袭血脑屏障的机制与高级体外血脑屏障模型（微流控/器官芯片）构建

## Inclusion

- 机制类：深入探讨隐球菌（或相关重要真菌/病原体）跨越血脑屏障的具体分子机制（如受体结合、紧密连接破坏、特洛伊木马机制等）。
- 模型类：详细阐述高级体外血脑屏障模型（如微生理系统、器官芯片、微流控系统、诱导多能干细胞(iPSC)衍生模型等）的构建方法、参数设置及功能评估。

## Exclusion

- 仅讨论隐球菌肺部感染或其他非中枢神经系统（CNS）感染。
- 采用传统常规的二维静态细胞培养，且无明显方法学突破或仿生设计的文章。
- 缺乏深层机制探讨的纯临床病例报告、流行病学统计或常规药物疗效对比。
