# Topic

数字设备使用与学生福祉（Digital device use and well-being among students）

## Inclusion

- 研究对象必须是学生群体（如青少年、中学生、大学生等）。
- 自变量必须包含明确的数字设备或媒介使用行为（如智能手机成瘾、社交媒体使用频率、整体屏幕时间）。
- 因变量必须涉及福祉的具体测量（如抑郁、焦虑、睡眠质量、孤独感、主观幸福感或学业倦怠）。
- 必须是高质量的实证研究（定量、定性或混合方法）或系统性综述/元分析。

## Exclusion

- 纯探讨数字设备硬件技术、通信协议或底层算法（无人类心理/行为测量）。
- 研究对象主要为非学生群体（如学龄前婴幼儿、老年人、全职职场人士）。
- 仅探讨数字设备在特定疾病临床治疗中的应用（如利用VR技术治疗特定恐惧症），而非日常使用习惯对普遍福祉的影响。
