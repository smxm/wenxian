# Topic

共价有机框架（COFs）在光热驱动海水蒸发中的作用及应用机制

## Inclusion

- 必须明确以共价有机框架 COFs 或其复合材料为核心研究对象
- 必须应用于光热转换、太阳能海水淡化、界面水蒸发或汽水发生器
- 内容需涉及分子结构设计 如带隙调控 、光热转换机制、水传输路径、或抗盐结晶性能。

## Exclusion

- 纯粹探讨 COF 合成或将其用于与光热蒸发无关的领域 如单纯的气体吸附、电池储能、常规催化
- 光热蒸发的主体材料为非 COF 如纯碳基材料、MOFs、气凝胶 且未与 COF 结合
- 缺乏具体性能数据 如蒸发速率 kg m-2 h-1 的低质量文章
