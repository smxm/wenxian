# Topic

理工科（STEM）学术写作中模糊语（Hedging / Vague Language）的使用及语用分析

## Inclusion

- 必须探讨学术写作、科技语篇或理工科 含医学、工程等硬学科 背景下的模糊语 hedging, vague language, epistemic modality 等 现象
- 包含语用功能分析、基于语料库的对比 如中外学者对比、学科对比 或科技文本翻译中的模糊语处理。

## Exclusion

- 严格排除金融和经济领域的“对冲 hedging ”
- 严格排除数学、计算机算法和控制系统中的“模糊逻辑/模糊控制 fuzzy logic/fuzzy control ”
- 排除非学术语境 如日常会话、小说、新闻 的分析。
