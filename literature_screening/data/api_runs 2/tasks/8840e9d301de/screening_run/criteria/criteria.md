# Topic

企业/高管风险承担（以研发支出比率作为测度指标）

## Inclusion

- 必须涉及“风险承担”（Risk-taking，特别是高管风险承担或企业风险承担）这一核心概念。
- 必须明确将“研发支出比率”（如R&D投入、研发支出/主营业务收入等）作为衡量风险承担的代理变量/测度指标。
- 优先保留包含类似实证框架（如：某变量 -> 风险承担(研发比率) -> 企业绩效等）的实证研究文献，或专门论述此测量方法的经典文献。

## Exclusion

- 探讨“研发活动本身的风险”或“如何降低研发风险”的文献（概念混淆）。
- 纯粹研究技术创新、专利产出，通篇未将其提升至“管理层风险承担”维度的文献。
- 纯算法、纯工程类或无实证数据支撑的非经管类文献。
