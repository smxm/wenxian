# Topic

人工智能（AI）与扩展现实（XR）技术在猫咪及动物交互中的应用（涵盖多模态感知、智能干预、XR界面与数字孪生设备）

## Inclusion

- 研究对象必须涉及猫咪（Cat/Feline）、伴侣动物（Companion Animals）或虚拟宠物（Virtual Pets）。
- 核心技术必须明确包含人工智能（AI、机器学习、计算机视觉、多模态情感计算等）或扩展现实/物联网（XR、VR、AR、数字孪生、智能穿戴设备等）。
- 研究焦点落在“交互”或“干预”上（如行为识别、无聊干预、跨物种界面设计、人宠互动）。

## Exclusion

- 绝对排除医学影像与临床医学文献（如涉及正电子发射断层扫描 PET scans、PET-CT、PET-MRI、阿茨海默症、肿瘤学）。
- 绝对排除化工材料文献（如涉及聚对苯二甲酸乙二醇酯 PET 塑料）。
- 排除纯兽医基础科学、纯生物学、传统畜牧业研究，且无现代计算机技术（AI/XR/IoT）深度参与的文章。
- 排除纯算法理论或纯硬件底层开发，未结合具体动物交互场景的文章。
