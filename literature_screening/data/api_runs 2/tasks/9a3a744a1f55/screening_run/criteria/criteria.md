# Topic

AI虚拟主播与虚拟商品类型对消费者购买意愿的影响

## Inclusion

- 必须涉及 AI 虚拟主播（Virtual Streamer/Influencer/Avatar）、数字人或虚拟代言人。
- 必须探讨消费者行为或心理学指标，特别是购买意愿（Purchase Intention）或感知价值。
- 研究背景最好包含虚拟电商、直播带货，或明确探讨了虚拟商品/数字产品（Virtual Goods/Digital Products）的分类与属性。

## Exclusion

- 纯计算机视觉、图形渲染、大模型底层算法等纯技术类文献（无消费者实证数据）。
- 与电子商务、市场营销完全无关的场景（如：纯医学辅助数字人、工业孪生数字人）。
- 非学术期刊/会议的无深度新闻报道。
