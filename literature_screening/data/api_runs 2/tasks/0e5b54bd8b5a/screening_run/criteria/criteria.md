# Topic

下一代益生菌（Akkermansia muciniphila/AKK）、发酵工艺及乳制品应用

## Inclusion

- 必须涉及下一代益生菌（特别是 Akkermansia muciniphila）或其后生元/灭活形式。
- 必须涉及乳制品（如牛奶、酸奶、奶酪等）基质，或发酵工艺。
- 研究重点包括但不限于：AKK在乳制品中的存活率提升（如微胶囊化、共培养）、后生元在乳制品中的理化特性影响、合生元开发等。

## Exclusion

- 纯医学/临床研究，仅探讨AKK对某种疾病（如糖尿病、肠炎）的治疗机制，完全未涉及食品基质或乳制品递送。
- 仅针对传统益生菌（如乳酸杆菌、双歧杆菌）的研究，未提及AKK或NGPs。
- 纯动物模型肠道菌群测序分析，无食品工程、发酵工艺或食品组分介入。
