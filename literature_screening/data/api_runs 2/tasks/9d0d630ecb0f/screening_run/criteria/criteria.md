# Topic

自身炎症及自身免疫性疾病（重点包含VKH、SLE等）的眼底及其他眼部表现

## Inclusion

- 全身性疾病背景：必须涉及自身炎症性疾病（Autoinflammatory）或自身免疫性疾病（Autoimmune），特别是明确提及 VKH（小柳原田综合征）、SLE（系统性红斑狼疮）或其他相关免疫介导疾病。
- 临床表型：必须探讨上述疾病引起的眼部受累表现。重点关注眼底、葡萄膜、视网膜、脉络膜或视神经的病变（如全葡萄膜炎、视网膜血管炎、渗出性视网膜脱离、棉絮斑等）。

## Exclusion

- 纯全身性免疫疾病研究，完全没有提及眼部/眼底表现的文献。
- 纯眼科局部疾病，缺乏全身性自身免疫/自身炎症背景的文献（例如：年龄相关性黄斑变性、糖尿病视网膜病变、单纯的机械性眼外伤等）。
- 仅作为流行病学数据一笔带过，缺乏对眼部特征、机制或影像学进行实质性探讨的低信息量文献。
