# Topic

克雷伯氏菌（*Klebsiella*）中甘油途径的高效表达与代谢工程改造

## Inclusion

- 研究对象涉及克雷伯氏菌 如 *Klebsiella pneumoniae*, *Klebsiella oxytoca* 等 。
- 核心围绕甘油代谢途径（如 1,3-丙二醇、3-羟基丙酸的合成）。
- 内容包含提高表达或产量的具体手段（如关键酶过表达、启动子/RBS优化、底盘细胞基因敲除、辅酶平衡、发酵工艺控制等）。

## Exclusion

- 纯临床医学/病理学文献 如克雷伯氏菌的致病机理、耐药性分析、医院感染控制 。
- 非甘油碳源的代谢研究（甘油仅作为背景或溶剂提及）。
- 缺乏实质性生物学改造或工艺优化数据的纯宏观综述（若为极高质量的专门综述可保留）。
