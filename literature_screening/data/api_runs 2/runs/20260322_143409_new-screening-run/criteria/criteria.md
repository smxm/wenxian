# Topic

BALF基线病毒谱经免疫调控影响SCAP患者预后的机制及预测研究

## Inclusion

- 疾病对象：必须涉及 重症社区获得性肺炎 (SCAP)、重症肺炎、脓毒症 (Sepsis) 或 ARDS。
- 核心要素（满足其一即可）：
- 样本/技术：涉及 肺泡灌洗液 (BALF) 且使用了 mNGS/tNGS 或病毒谱分析。
- 病原学：讨论了 病毒感染（含混合感染、病毒再激活，如CMV/EBV/流感/腺病毒）。
- 机制/指标：涉及 免疫调控、细胞因子风暴、炎症指标（如IL-6, IL-10, SAA, 淋巴细胞亚群）。
- 结局：涉及 预后分析、死亡率预测、预测模型构建。
- 人群：优先成人，但如果儿童研究涉及高质量的“mNGS应用”或“免疫机制/预测模型”，也请纳入（需在分析中注明是儿科）。

## Exclusion

- 纯护理类文章（无生理/病理机制探讨）。
- 纯药物疗效对比（无免疫/病原学深度分析）。
- 与肺部感染/重症医学完全无关的文献。
