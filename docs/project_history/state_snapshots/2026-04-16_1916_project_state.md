- `/threads/new` now only creates a thread from the research need; follow-up work starts after the thread exists.
- The thread detail page is the main control surface: one edit entry at the top, plus four stage actions for search strategy, screening, unified review, and report generation.
- The unified review area is a project-level candidate workbench with separate round context, stable candidate identities, batch/single-item status updates, and report-source decisions driven by final inclusion.
